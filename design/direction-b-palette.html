Sync engine — design notes

Goal: multi-device sync that never surprises the person typing. The invariant everything else hangs off: the file on disk is always a valid document, and the [[operation log]] is always sufficient to rebuild it.

Every mutation appends to the log before it touches state. Entries are JSONL with a hash chain — each entry carries the hash of its predecessor, so a torn write is detectable at the exact line it happened.
